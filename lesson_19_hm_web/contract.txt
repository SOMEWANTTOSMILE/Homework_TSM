1.Приложение создает базу данных, а в ней таблицу со следующими колонками:
    1. id - id пользователя и его прием пищи, он должен быть первичным ключом
    2. user - имя пользователя, оно должно быть строковым и не быть пустым
    3. food - название еды, оно должно быть числовым и не должно ровняться нулю
    4. weight - масса еды, числовое и не равное нулю
    5.proteins, fats, carbohydrates - белки, жиры и углеводы, должны быть числовыми и не равняющие нулю
2. Сайт должен принимать и отдавать данные(методы post и get). Страница сайта должны быть в формате html
   в котором должен выводить все данные по середине экрана, должны быть поля ввода данных: имя пользователя,
   название продукта, его масса. Должны быть поля ввода для белков, жиров и углеводов. Должны быть кнопка 
   которая отправляет данные на сервер.
3. приложение должно принимать все введенные данные о пользователе и добавлять в ранне созданную базу данных
   с колонками описанными выше. Командная строка должна выводить в виде списка с ключом и значением все введенные
   пользователем данные
